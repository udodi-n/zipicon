# Zip.Icons

A ocial media icons downloader. Type the icons you want, hit download, get a zip.

## What it does

You come in, type the names of the social icons you want (separated by commas), and Zip.Icons bundles them all into a `.zip` file and sends it straight to your browser. No accounts. No drag and drop. No browsing through endless icon packs.

Just type. Download. Done.

## How to use it

1. Visit the app
2. Type the names of the icons you want, separated by commas
   ```
   facebook, twitter, instagram, linkedin
   ```
3. Hit download
4. Get a `.zip` with all your icons as `.png` files

## Icons

Icons are sourced from [Simple Icons](https://simpleicons.org/) and served as PNGs.

## Tech Stack

- **Frontend** — React
- **Backend** — Node.js
- **Icons** — Simple Icons + Sharp (for PNG conversion)
